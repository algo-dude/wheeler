About IBKR Integration

Wheeler connects to Interactive Brokers TWS (Trader Workstation) or IB Gateway to automatically sync your positions.

  • TWS Paper Trading: Port 7497 (default)
  • TWS Live Trading: Port 7496
  • IB Gateway Paper: Port 4002
  • IB Gateway Live: Port 4001

Requirements:

  • TWS or IB Gateway must be running and logged in
  • API connections must be enabled in TWS/Gateway settings
  • If using Docker, ensure network connectivity to TWS host

The sync will import all stock and option positions from your IBKR account into Wheeler for tracking and analysis.
